# Admin Dashboard

A modern admin dashboard built with **HTML5** and **CSS3** as part of **The Odin Project Foundations** curriculum.

## Features
 
- **Grid-based layout** — outer page grid (sidebar / header / main) with nested grids inside the sidebar, header, project cards, announcements, and trending sections.
- **Responsive** — collapses to a single column below 900px.
- **Live search** — filters project cards by title/description as you type.
- **New project** — modal form to create a project card on the fly, prepended to the grid.
- **Upload** — triggers a native file picker and confirms the upload with a toast.
- **Share** — uses the Web Share API where supported, falling back to copying the page link to your clipboard.
- **Per-card actions** — favorite (star), view, and share icons on each project card give toast feedback.
