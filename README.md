# Admin Dashboard

A modern admin dashboard built with **HTML5** and **CSS3** as part of **The Odin Project Foundations** curriculum.

## Features

- CSS Grid-based layout
- Responsive dashboard structure
- Sidebar navigation
- Header with search and user profile
- Project cards
- Announcements section
- Trending users section
- Material Design Icons
- Google Fonts integration

## Built With

- HTML5
- CSS3
- CSS Grid
- Flexbox
- Material Design Icons
- Google Fonts

## Skills Practiced

- CSS Grid
- Nested grids
- Layout design
- Responsive UI principles
- Semantic HTML
- Card-based interfaces
